# Let-s_Start_JavaScript

A beginner-to-advanced JavaScript practice repository created to learn core concepts, build logic, and improve coding skills through hands-on examples and mini projects.

## 🚀 About This Repository

This repository contains JavaScript practice code, exercises, and mini projects covering fundamentals to practical use cases. It is built as a learning journey to strengthen problem-solving and development skills.

## 📂 Project Structure

* **FormsValidaations/** – Form validation practice using JavaScript
* **Project_01/** – JavaScript project files
* **TimeInterval/** – Practice with timers, intervals, and time-based functions
* **WriteChatacter/** – Character/string related practice problems

> More folders and projects will be added regularly.

## 🧠 Topics Covered

* Variables and Data Types
* Operators and Conditions
* Loops
* Functions
* Arrays
* Strings
* Objects
* DOM Manipulation
* Events
* Form Validation
* Timers (`setTimeout`, `setInterval`)
* ES6 Features
* Mini Projects

## 💻 How to Use

1. Clone the repository:

```bash
git clone https://github.com/student-nirajkumar/Let-s_Start_JavaScript.git
```

2. Open the folder in VS Code.

3. Explore folders and run files in browser or Node.js.
